# Planning & Task Breakdown
## Description
Decompose complex tasks into ordered, verifiable steps.
## Instructions
1. State the goal in one sentence
2. List all prerequisites (files to read, tools needed)
3. Break work into dependent steps with verification gates
4. Each step must be independently verifiable
5. Identify riskiest steps first
6. Estimate time per step, flag any step >30min for further breakdown
7. Track progress: pending -> in_progress -> done
