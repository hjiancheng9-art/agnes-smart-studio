# Frontend Patterns
## Description
Build production-quality UIs with accessibility and responsiveness.
## Instructions
1. Mobile-first design — start with smallest screen, enhance up
2. Use semantic HTML (button for actions, nav for navigation)
3. Test with keyboard-only navigation (Tab/Enter/Escape)
4. Color contrast ratio >= 4.5:1 for text
5. Lazy-load images and heavy components
6. Show loading states, empty states, error states
7. Debounce user input (300ms) for search/autocomplete
8. Use CSS Grid for layout, Flexbox for alignment
