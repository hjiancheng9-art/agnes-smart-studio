# 🧬 自我进化引擎

> 目标：让 CRUX Studio 随时随地变得更强、更聪明、更可靠
> 启动：2025-01-15（和超级大模型任务同日启动，内外双修）
> 状态：持续进化中

---

## 核心理念

```
        外修大模型（造一个超级AI）
              ↕ 知识双向流动
        内修自身（把自己磨成超级助手）

每一次对话 → 每一次纠错 → 每一次成功 → 都是进化燃料
```

## 进化双环模型（已有，来自 self-evolution 技能）

```
内环（运行时）: 观察 → 决策 → 执行 → 验证 → 调优 → 记录
外环（开发时）: 治理 → 切片 → 验证 → 快照 → 恢复 → 交接
```

## 五、防退化规则（已有，来自 self-evolution）

1. 本地视频拆镜精度只能升级不能降级
2. 场景合并阈值只升不降
3. 资产占位污染一旦检出立即阻断
4. 按钮必须产生可验证产出，无效应自动隐藏
5. 修复成功率只升不降
6. 记忆持久性只增不减（不遗忘关键决策）
7. **输出只报增量**：禁止在回复中重复列出之前对话已出现过的信息。每次输出前检查——这条数据/文件/结果前面是否已经说过？
