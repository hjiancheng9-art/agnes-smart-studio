# 命令参考

聊天模式 (`agnes_studio.py -c`) 下可用的 25 个命令：

| 命令 | 功能 | 示例 |
|------|------|------|
| `/help` | 显示帮助 | /help |
| `/model light\|pro` | 切换模型 | /model pro |
| `/thinking` | 深度思考开关 | /thinking |
| `/code` | 编程助手模式 | /code |
| `/agent` | 智能体模式 | /agent |
| `/plan <任务>` | 先规划再执行 | /plan 写一个爬虫 |
| `/sub <任务>` | 子智能体 | /sub 检查安全漏洞 |
| `/compress` | 压缩对话历史 | /compress |
| `/img <描述>` | 生成图片 | /img 一只猫 |
| `/video <描述>` | 生成视频 | /video 日落海边 |
| `/vision <图> <问>` | 图片理解 | /vision photo.jpg 描述画面 |
| `/project [cmd]` | 项目管理 | /project save my-app |
| `/team [类型]` | 智能体团队 | /team review |
| `/deploy [目标]` | 一键部署 | /deploy vercel |
| `/todo [path]` | TODO扫描 | /todo |
| `/commit` | 自动commit | /commit |
| `/changelog` | 生成CHANGELOG | /changelog |
| `/refactor <旧> <新>` | 批量重构 | /refactor old new . |
| `/audit [pip\|npm]` | 依赖审计 | /audit |
| `/rules [cmd]` | 规则管理 | /rules enable python-style |
| `/automate [cmd]` | 自动化任务 | /automate add 检查 "0 9 * * 1" |
| `/provider [cmd]` | 模型供应商 | /provider switch kimi |
| `/evolve` | 进化状态 | /evolve |
| `/know [cmd]` | 知识库 | /know methods |
| `/skill [cmd]` | 技能管理 | /skill load cinematic-master |
| `/tools` | 工具列表 | /tools |
| `/self [cmd]` | 自诊断 | /self check |
| `/clear` | 清空历史 | /clear |
| `/exit` | 退出 | /exit |
