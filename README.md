# CRUX Studio v6.0.0

**AI 驱动的全栈开发工作站** — 智能体编排、代码生成、浏览器自动化。

## 核心能力

- 多模型 AI 推理（DeepSeek / Agnes / CodeBuddy）
- Playwright CDP 浏览器全操控
- 图片/视频/语音生成
- GitHub 集成、Git 工作流
- MCP 工具网格（TRM — 七兽路由）
- TDD / Code Review / 自我修复
- ComfyUI 工作流编译（通过 ComfyFlow Compiler）

## 快速启动

```bash
crux              # 交互模式
crux --help       # 查看所有命令
```

## 项目结构

| 目录 | 说明 |
|------|------|
| `core/` | 聊天会话、CLI处理 |
| `skills/` | 技能市场（56+ 技能包） |
| `tools/` | 工具函数 |
| `ui/` | 终端UI |
| `agents/` | 智能体编排 |
| `docs/` | 文档 |
