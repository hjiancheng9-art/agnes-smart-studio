## 五分钟上手

### 1. 安装

```bash
pip install -r requirements.txt
cp .env.example .env   # 填入你的 API Key
```

### 2. 启动

```bash
python crux_studio.py          # 交互模式
python crux_studio.py -q "你的问题"  # 快速模式
```

### 3. 常用命令

| 命令 | 用途 |
|------|------|
| `/help` | 查看所有命令 |
| `/skill list` | 列出可用技能 |
| `/model` | 切换/查看模型 |
| `/img` 描述 | 生成图片 |
| `/video` 描述 | 生成视频 |
| `/git commit` | 提交代码 |
| `/todo` | 任务追踪 |
| `/plan` | 多步规划执行 |
| `/audit` | 代码审查 |
| `/self heal` | 系统自检修复 |

### 4. 权限模式

```
/yolo    → 所有工具直接执行（信任模式）
/auto    → 高危工具需确认（默认）
/manual  → 所有写入操作需确认（安全模式）
```

### 5. 核心能力索引

```

/朱雀    → 代码审查
```
