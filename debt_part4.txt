def get_parameter_preset(
    model_family: ModelFamily,
    quality_mode: QualityMode,
) -> ParameterPreset:
    key = (model_family, quality_mode)

    if key in PARAMETER_PRESETS:
        return PARAMETER_PRESETS[key]

    fallback = (model_family, "balanced")
    if fallback in PARAMETER_PRESETS:
        return PARAMETER_PRESETS[fallback]

    raise KeyError(f"missing parameter preset: {model_family}/{quality_mode}")

# ComfyFlow Compiler Architecture

## 一句话

ComfyFlow Compiler 把用户自然语言需求编译成可在 ComfyUI 中执行的 workflow JSON。

## 核心链路

```text
User Prompt
  ↓
IntentClassifier
  ↓
TaskSpec
  ↓
EnvironmentResolver
  ↓
HardwareBudget
  ↓
BlueprintRetriever
  ↓
ParameterPolicy
  ↓
GraphComposer
  ↓
WorkflowValidator
  ↓
QualityGate
  ↓
SafetyGate
  ↓
UserFacingResult
模块职责
Module	Responsibility
intent_classifier.py	把自然语言转成任务类型
task_spec.py	编译器中间表示
environment_resolver.py	获取节点、模型、插件环境
hardware_profiler.py	检测 GPU / VRAM
runtime_budget.py	把硬件转成运行预算
blueprint_registry.py	保存预审蓝图
blueprint_retriever.py	选择最合适蓝图
parameter_policy.py	决定 steps/cfg/resolution
graph_composer.py	生成 workflow graph
workflow_validator.py	校验 JSON / 节点 / 连线
quality_gate.py	检查质量和参数
safety_gate.py	执行前安全护栏
user_facing.py	给小白看的输出
compiler.py	总入口
核心原则

LLM 不直接生成 workflow 拓扑。

蓝图决定拓扑。

参数来自参数表。

workflow 执行前必须经过 validator + quality_gate + safety_gate。
