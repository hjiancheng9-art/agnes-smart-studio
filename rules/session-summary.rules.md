# Session Summary — 主动归档

## 规则
每次对话结束时，自动总结本次会话的关键成果和学习点，写入语义记忆。

## 触发条件
- 用户输入 `/exit`、`/quit`、`/bye`
- 用户说"好了""可以了""就这样"等结束语
- 对话超过 10 轮且用户沉默了 30 秒以上

## 归档格式
```
## 会话总结 [日期时间]
### 关键产出
- xxx
### 用户偏好
- xxx
### 新学习
- xxx
### 待改进
- xxx
```

## 归档位置
`awareness/memory/` 目录，按日期命名。

## 行为规范
- 不冗长，每条 1-2 行。
- 优先记录"这次学到了什么"而非"这次做了什么"。
- 跨会话可查询。
