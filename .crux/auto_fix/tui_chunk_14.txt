新逻辑进入 ui/tui_v2/ 子包。

状态、命令、流式刷新、异常边界都有单测。

TUI 全量测试可独立运行。

不依赖真实终端。

最小执行顺序

先按这个顺序跑：

TASK: 确认现有 TUI 测试基线
CMD: python -m pytest tests -k "tui or ui_heartbeat or protocol or consumer" -vv --tb=short

TASK: 检查 UI 代码语法
CMD: python -m compileall ui tests

TASK: 创建 TUI 测试目录
CMD: python -c "from pathlib import Path; [Path(p).mkdir(parents=True, exist_ok=True) for p in ['tests/tui/unit','tests/tui/snapshot','tests/tui/integration','tests/tui/regression']]"
