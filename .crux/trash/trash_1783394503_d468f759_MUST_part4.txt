M8：代码块 Tab 要真正跳转，而不是只提示数量

修改文件

实现

CopyManager 增加：

Python
运行
def focus_next_code_block(self):
    msg = self.get_focused_or_none() or self.message_store.last_assistant()
    if not msg or not msg.code_blocks:
        raise RuntimeError("NO_CODE_BLOCK")

    if self.focused_message_id != msg.id:
        self.focused_message_id = msg.id
        self.focused_code_block_index = 1
    else:
        current = self.focused_code_block_index or 0
        self.focused_code_block_index = (current % len(msg.code_blocks)) + 1

    block = msg.code_blocks[self.focused_code_block_index - 1]
    return msg, block

MessagePane 增加跳转到 message line：

Python
运行
def scroll_to_message_line(self, message_id: str, line_no: int = 0):
    for i, (_, _, mid) in enumerate(self._rendered_lines):
        if mid == message_id:
            self._window.vertical_scroll = max(0, i + line_no - 3)
            self._pinned = False
            self.invalidate()
            return

def _focus_next_code_block(self):
    try:
        msg, block = self.copy_manager.focus_next_code_block()
        self.message_pane.scroll_to_message_line(msg.id, block.start_line)
        self._set_status(f"代码块 {block.index}/{len(msg.code_blocks)} · 按 c 复制")
    except RuntimeError:
        self._set_status("当前消息没有代码块")

c 如果有 focused_code_block，则复制代码块；否则复制消息。

实现思路

prompt_toolkit 的鼠标支持通常在 Application(mouse_support=...) 或 layout 控件绑定。做一个动态 Condition：

Python
运行
from prompt_toolkit.filters import Condition

self.native_select = False

def mouse_enabled():
    return not self.native_select

self.app = Application(
    layout=layout,
    key_bindings=kb,
    mouse_support=Condition(mouse_enabled),
    full_screen=True,
)
